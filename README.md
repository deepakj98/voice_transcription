# voice_transcription
Real-time transcription, where a user can visit a page, click a button to start recording their voice, see a real-time transcription on the web page, and then after clicking stop, see the full transcription and a summary of the conversation.

1. Clone the project 
	`git clone https://github.com/deepakj98/voice_transcription.git`
2. move to the project directory
	`cd voice_transcription`
3. run the bundler
	`bundle install`
4. create the database
	`rails db:create`
5. run the migration
	`rails db:migrate`
6. create a `.env` file and save there
	`OPENAI_API_KEY`
7. start the server
	`rails s`
